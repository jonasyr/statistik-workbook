# Project Overview: Statistik-Workbook WiSe 2025

## Purpose
This is an academic statistics workbook project for the Winter Semester 2025, analyzing the ALLBUS 2021 dataset (German General Social Survey). The project contains statistical analyses including:
- **Block 1**: Univariate descriptive statistics (scale levels, frequency tables, measures of central tendency)
- **Block 2**: Bivariate descriptive statistics (cross-tabulations, correlation)
- **Block 3**: Inferential statistics (confidence intervals, t-tests)
- **Block 4**: Correlation and regression analysis
- **Block 5**: One-way ANOVA

The final deliverable is a LaTeX-formatted workbook documenting all analyses.

## Programming Language
**Python** - all analysis scripts are written in Python

## Main Datasets
- ALLBUS 2021 subset (29 variables)
- Located in `data/raw/`
- Processed data saved to `data/processed/`
- Variables include demographic data (age, sex, income) and survey responses

## Project Type
Academic/educational - statistics course assignment with strict task specifications in German
